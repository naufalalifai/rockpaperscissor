## Description
This project is a Rock–Paper–Scissor multiplayer game.

## Features
- 2-player multiplayer game
- Client–server architecture using TCP sockets
- Best-of-3 rounds
- 10-second timeout for player moves
- GUI client using Tkinter
- Email notification to the game winner using SMTP

## How to Run

### 1. Set Email Credentials
Create a `.env` file (not committed to GitHub) with:

> RPS_EMAIL=your_email@gmail.com
> <br> RPS_EMAIL_PASS=your_app_password


### 2. Run the Server
> python server.py

### 3. Run the Clients (2 terminals)
> python client.py
